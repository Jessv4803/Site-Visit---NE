How to set up ▾
1. Open your Google Sheet → Extensions → Apps Script
2. Delete any existing code, paste the SiteVisit_NE_AppScript.gs script, click Save 💾
3. Run setupSheets() once (click the function dropdown → Run) to create the 4 tabs
4. Deploy → New deployment → Web app
5. Set Execute as: Me · Who has access: Anyone
6. Click Deploy and authorize when prompted
7. Copy the Web App URL ending in /exec and paste below
Must start with https://script.google.com/macros/s/ and end with /exec
